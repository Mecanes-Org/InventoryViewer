# InventoryViewer
The Inventory Viewer module is a tool designed to enhance the user experience for Pro Inventory System users. It provides a fast and intuitive way to view their inventory in real-time during various actions, such as adding, setting, or removing items.

## Description

Inventory Viewer simplifies inventory management by providing a clear and instant view of its contents. This module integrates directly into your workflow, offering a better understanding of any changes made to the inventory.

## Objective

Its main goal is to allow developers and end-users to monitor and interact with their inventory without interrupting the game flow or development process.

## Requirement : Pro Inventory System

**Target version** : **UE5.3 ～ 5.5**

**Pro Inventory System** already installed and configured.

## Warning

- This module is designed to work exclusively with Pro Inventory System.

## Installation

- Install Pro Inventory System via the marketplace .
- Please make sure the plugin is enabled by going to Edit > Plugins.
- Go to Edit > Plugins.
- Go to your unreal project
- **Plugins > ProInventorySystem > Source**
- Paste the ‘EditorInventoryViewer’ folder

## Advantages and uses

* **Speed**: Instant visualization of inventory changes.
* **Clarity**: An intuitive and easy-to-use user interface.
* **Efficiency**: Reduces the time required to check or debug inventory changes.
* **Speed**: Compatible with custom inventory systems built with **Pro Inventory System.**

## Features & applications

* **Real-time preview of added, modified, or removed items.**
* **Dynamic updates during inventory interactions.**
